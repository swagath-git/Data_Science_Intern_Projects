# Data Science Projects

This repository contains multiple data science projects implemented using Python and Jupyter Notebook. Each project includes data preprocessing, exploratory data analysis (EDA), and basic model building.

---

## 📂 Projects Included

### 1. Customer Churn Analysis (20% Dataset)
- Loaded dataset and performed initial exploration
- Handled missing values
- Encoded categorical features using LabelEncoder
- Performed EDA using heatmaps, histograms, and boxplots
- Split data into training and testing sets

---

### 2. Customer Churn Analysis (80% Dataset)
- Loaded and explored dataset
- Removed missing values
- Encoded categorical columns
- Applied feature scaling using StandardScaler
- Performed basic visualizations (boxplot, histogram, scatterplot)

---

### 3. House Price Prediction
- Loaded housing dataset
- Handled missing values
- Encoded categorical variables
- Performed correlation analysis using heatmap
- Generated statistical summary of dataset

---

### 4. Iris Dataset Analysis
- Loaded iris dataset
- Handled missing values
- Encoded target variable (species)
- Applied feature scaling
- Performed EDA using boxplots, histograms, and scatterplots

---

### 5. Sentiment Analysis
- Loaded sentiment dataset
- Encoded sentiment labels
- Performed basic EDA (countplot)
- Converted text data into numerical format using TF-IDF
- Split data into training and testing sets
- Trained Logistic Regression model
- Evaluated model using accuracy and classification report
